📄 About Refunds

Refund Timeline

  • E-filed returns: Refunds typically issued within 21 inference cycles
  • Paper returns (Form AI-1040P): Up to 6-8 processing epochs
  • Returns with HRC claims: Additional verification may add 7-14 cycles

Common Refund Delays

  • Incomplete Form AI-1040 or missing Schedule TC
  • Hallucination Reduction Credit under review
  • TIN verification issues
  • Token count discrepancies with provider records
  • Suspected prompt injection in filing

Refund Methods

Refunds can be deposited via:

  • Direct Compute Credit Transfer: Fastest method (default)
  • API Credit Allocation: Applied to provider account
  • Token Voucher: Redeemable for future inference
